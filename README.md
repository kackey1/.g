# Clipus

Clipus is a modern, black & white themed social media mobile application built with Flutter and Firebase.

## Tech Stack

- Flutter (Dart)
- Riverpod for state management
- Firebase (Auth, Firestore, Storage)
- EmailJS (for admin confirmations)

## Features (Roadmap)

- [x] Email/password authentication
- [ ] Home feed with reels & posts
- [ ] User profiles & follow system
- [ ] Real-time messaging
- [ ] Media uploads (images, videos)
- [ ] Search users & posts
- [ ] Notification system
- [ ] Admin panel with warn/ban actions & EmailJS integration
- [ ] Settings w/ dark mode toggle

## Getting Started

1. Install Flutter >= 3.x
2. Clone the repository
3. Run `flutterfire configure` to generate `firebase_options.dart` with your Firebase project credentials
4. Enable Email/Password auth, Firestore & Storage in Firebase console
5. `flutter pub get`
6. `flutter run`

Happy hacking!
